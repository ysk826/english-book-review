## メモ

- タスクを着手する前に、必ず Claude Code に Planning モードで方針を出してもらうこと。

- 完了時には、必ずテストと Lint をパスさせてから Done に移動させること。
- 仕様の変更が発生した場合は、TASKS.md ではなく SPEC.md を先に更新すること。
